value_get_shader_matrix

@accepts(GObject.Value, gsize)
@returns([gfloat])
def value_get_shader_matrix(value, length)

Retrieves a matrix of floating point values stored inside the passed GValue. @value must have been initialized with %CLUTTER_TYPE_SHADER_MATRIX.

of floating point values. The returned value is owned by the GValue and should never be modified or freed.

value :

a GValue

length :

return location for the number of returned floating point values, or None

Returns :

the pointer to a matrix

Since 0.8
