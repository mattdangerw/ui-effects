get_press_coords

@accepts(guint, gfloat, gfloat)
@returns(none)
def get_press_coords(device, press_x, press_y)

Retrieves the coordinates, in stage space, of the press event that started the dragging for an specific pointer device

device :

currently unused, set to 0

press_x :

return location for the press event's X coordinate

press_y :

return location for the press event's Y coordinate

Since 1.8
