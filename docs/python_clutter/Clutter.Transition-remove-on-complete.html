Clutter.Transition:remove-on-complete

"remove-on-complete"             gboolean                : Read / Write

Whether the Clutter.Transition should be automatically detached from the Clutter.Transition:animatable instance whenever the Clutter.Timeline::completed signal is emitted.

The Clutter.Transition:remove-on-complete property takes into account the value of the Clutter.Timeline:repeat-count property, and it only detaches the transition if the transition is not repeating.
