add_effect_with_name

@accepts(utf8, Clutter.Effect)
@returns(none)
def add_effect_with_name(name, effect)

A convenience function for setting the name of a Clutter.Effect while adding it to the list of effectss applied to @self

This function is the logical equivalent of:

|[ clutter_actor_meta_set_name (CLUTTER_ACTOR_META (effect), name); clutter_actor_add_effect (self, effect); ]|

name :

the name to set on the effect

effect :

a Clutter.Effect

Since 1.4
