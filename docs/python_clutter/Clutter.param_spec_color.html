param_spec_color

@accepts(utf8, utf8, utf8, Clutter.Color, GObject.ParamFlags)
@returns(GObject.ParamSpec)
def param_spec_color(name, nick, blurb, default_value, flags)

Creates a GParamSpec for properties using Clutter.Color.

name :

name of the property

nick :

short name

blurb :

description (can be translatable)

default_value :

default value

flags :

flags for the param spec

Returns :

the newly created GParamSpec
