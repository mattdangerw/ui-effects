Clutter.DeviceManager::device-added

def callback(device_manager, device, user_param1, ...)

The ::device-added signal is emitted each time a device has been added to the Clutter.DeviceManager

device_manager :

instance of Clutter.DeviceManager that is emitting the signal

device :

the newly added Clutter.InputDevice

user_param1 :

first user parameter (if any) specified with the connect() method

... :

additional user parameters (if any)

Since 1.2
