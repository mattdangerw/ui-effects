Clutter.LayoutMeta

Sub-class of Clutter.ChildMeta specific for layout managers

A Clutter.LayoutManager sub-class should create a Clutter.LayoutMeta instance by overriding the Clutter.LayoutManager::create_child_meta() virtual function

from gi.repository import Clutter

layout_meta = Clutter.LayoutMeta(manager=value)
