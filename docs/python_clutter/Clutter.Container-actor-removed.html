Clutter.Container::actor-removed

def callback(container, actor, user_param1, ...)

The ::actor-removed signal is emitted each time an actor is removed from @container.

container :

instance of Clutter.Container that is emitting the signal

actor :

the child that has been removed from @container

user_param1 :

first user parameter (if any) specified with the connect() method

... :

additional user parameters (if any)

Since 0.4
