ClutterTouchEvent

Used for touch events. The @type field will be one of %CLUTTER_TOUCH_BEGIN, %CLUTTER_TOUCH_END, %CLUTTER_TOUCH_UPDATE, or %CLUTTER_TOUCH_CANCEL. Touch events are grouped into sequences; each touch sequence will begin with a %CLUTTER_TOUCH_BEGIN event, progress with %CLUTTER_TOUCH_UPDATE events, and end either with a %CLUTTER_TOUCH_END event or with a %CLUTTER_TOUCH_CANCEL event. With multi-touch capable devices there can be multiple event sequence running at the same time.
