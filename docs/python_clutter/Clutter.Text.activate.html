activate

@returns(gboolean)
def activate()

Emits the Clutter.Text::activate signal, if @self has been set as activatable using clutter_text_set_activatable().

This function can be used to emit the ::activate signal inside a Clutter.Actor::captured-event or Clutter.Actor::key-press-event signal handlers before the default signal handler for the Clutter.Text is invoked.

and False otherwise

Returns :

True if the ::activate signal has been emitted,

Since 1.0
