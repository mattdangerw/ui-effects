Clutter.Animatable.get_initial_state

@accepts(utf8, GObject.Value)
@returns(none)
def do_get_initial_state(self, property_name, value):

Retrieves the current state of @property_name and sets @value with it

property_name :

the name of the animatable property to retrieve

value :

a GValue initialized to the type of the property to retrieve
