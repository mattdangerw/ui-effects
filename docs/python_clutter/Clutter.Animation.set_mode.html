set_mode

@accepts(gulong)
@returns(none)
def set_mode(mode)

Sets the animation @mode of @animation. The animation @mode is a logical id, either coming from the Clutter.AnimationMode enumeration or the return value of clutter_alpha_register_func().

This function will also set Clutter.Animation:alpha if needed.

mode :

an animation mode logical id

Since 1.0
