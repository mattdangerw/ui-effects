update_interval

@accepts(utf8, Clutter.Interval)
@returns(none)
def update_interval(property_name, interval)

Changes the @interval for @property_name. The Clutter.Animation will take ownership of the passed Clutter.Interval.

property_name :

name of the property

interval :

a Clutter.Interval
