set_attributes

@accepts(Pango.AttrList)
@returns(none)
def set_attributes(attrs)

Sets the attributes list that are going to be applied to the Clutter.Text contents.

The Clutter.Text actor will take a reference on the PangoAttrList passed to this function.

attrs :

a PangoAttrList or None to unset the attributes

Since 1.0
