set_data

@accepts([guint8], Cogl.PixelFormat, guint, guint, guint)
@returns(gboolean)
def set_data(data, pixel_format, width, height, row_stride)

Sets the image data to be display by @image.

If the image data was successfully loaded, the @image will be invalidated.

In case of error, the @error value will be set, and this function will return False.

The image data is copied in texture memory.

and False otherwise.

data :

the image data, as an array of bytes

pixel_format :

the Cogl pixel format of the image data

width :

the width of the image data

height :

the height of the image data

row_stride :

the length of each row inside @data

Returns :

True if the image data was successfully loaded,

Since 1.10
