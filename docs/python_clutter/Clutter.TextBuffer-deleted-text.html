Clutter.TextBuffer::deleted-text

def callback(text_buffer, position, n_chars, user_param1, ...)

This signal is emitted after text is deleted from the buffer.

text_buffer :

instance of Clutter.TextBuffer that is emitting the signal

position :

the position the text was deleted at.

n_chars :

The number of characters that were deleted.

user_param1 :

first user parameter (if any) specified with the connect() method

... :

additional user parameters (if any)

Since 1.10
