remove_key

@accepts(utf8, utf8, GObject.Object, utf8)
@returns(none)
def remove_key(source_state_name, target_state_name, object, property_name)

Removes all keys matching the search criteria passed in arguments.

source_state_name :

the source state name to query, or None for all source states

target_state_name :

the target state name to query, or None for all target states

object :

the specific object instance to list keys for, or None for all managed objects

property_name :

the property name to search for, or None for all properties.

Since 1.4
