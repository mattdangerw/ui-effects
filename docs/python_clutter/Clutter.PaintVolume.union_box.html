union_box

@accepts(Clutter.ActorBox)
@returns(none)
def union_box(box)

Unions the 2D region represented by @box to a Clutter.PaintVolume.

This function is similar to clutter_paint_volume_union(), but it is specific for 2D regions.

box :

a Clutter.ActorBox to union to @pv
