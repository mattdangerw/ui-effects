list_timelines

@returns(<list>)
def list_timelines()

Retrieves a list of all the ClutterTimelines managed by @score.

GSList containing all the timelines in the score. This function does not increase the reference count of the returned timelines. Use g_slist_free() on the returned list to deallocate its resources.

Returns :

a
