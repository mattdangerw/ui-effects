set_subtitle_font_name

@accepts(utf8)
@returns(none)
def set_subtitle_font_name(font_name)

Sets the font used by the subtitle renderer. The @font_name string must be either None, which means that the default font name of the underlying implementation will be used; or must follow the grammar recognized by pango_font_description_from_string() like:

|[ clutter_media_set_subtitle_font_name (media, "Sans 24pt"); ]|

font_name :

a font name, or None to set the default font name

Since 1.2
