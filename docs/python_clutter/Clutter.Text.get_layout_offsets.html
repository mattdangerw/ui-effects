get_layout_offsets

@accepts(gint, gint)
@returns(none)
def get_layout_offsets(x, y)

Obtains the coordinates where the Clutter.Text will draw the PangoLayout representing the text.

x :

location to store X offset of layout, or None

y :

location to store Y offset of layout, or None

Since 1.8
