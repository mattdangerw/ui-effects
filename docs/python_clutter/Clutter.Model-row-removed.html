Clutter.Model::row-removed

def callback(model, iter, user_param1, ...)

The ::row-removed signal is emitted when a row has been removed. The data on the row pointed by the passed iterator is still valid when the ::row-removed signal has been emitted.

model :

instance of Clutter.Model that is emitting the signal

iter :

a Clutter.ModelIter pointing to the removed row

user_param1 :

first user parameter (if any) specified with the connect() method

... :

additional user parameters (if any)

Since 0.6
