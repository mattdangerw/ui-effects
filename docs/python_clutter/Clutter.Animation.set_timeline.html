set_timeline

@accepts(Clutter.Timeline)
@returns(none)
def set_timeline(timeline)

Sets the Clutter.Timeline used by @animation.

This function will take a reference on the passed @timeline.

timeline :

a Clutter.Timeline, or None to unset the current Clutter.Timeline

Since 1.0
