Clutter.Texture::load-finished

def callback(texture, error, user_param1, ...)

The ::load-finished signal is emitted when a texture load has completed. If there was an error during loading, @error will be set, otherwise it will be None

texture :

instance of Clutter.Texture that is emitting the signal

error :

A set error, or None

user_param1 :

first user parameter (if any) specified with the connect() method

... :

additional user parameters (if any)

Since 1.0
