Clutter.BehaviourPath::knot-reached

def callback(behaviour_path, knot_num, user_param1, ...)

This signal is emitted each time a node defined inside the path is reached.

behaviour_path :

instance of Clutter.BehaviourPath that is emitting the signal

knot_num :

the index of the Clutter.Knot reached

user_param1 :

first user parameter (if any) specified with the connect() method

... :

additional user parameters (if any)

Since 0.2
