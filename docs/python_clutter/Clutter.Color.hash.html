hash

@accepts(gpointer)
@returns(guint)
def hash(v)

Converts a Clutter.Color to a hash value.

This function can be passed to g_hash_table_new() as the @hash_func parameter, when using Clutter.Color<!-- -->s as keys in a GHashTable.

v :

a Clutter.Color

Returns :

a hash value corresponding to the color

Since 1.0
