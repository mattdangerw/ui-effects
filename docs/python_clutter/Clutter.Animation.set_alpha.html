set_alpha

@accepts(Clutter.Alpha)
@returns(none)
def set_alpha(alpha)

Sets @alpha as the Clutter.Alpha used by @animation.

If @alpha is not None, the Clutter.Animation will take ownership of the Clutter.Alpha instance.

clutter_timeline_set_progress_mode() instead.

alpha :

a Clutter.Alpha, or None to unset the current Clutter.Alpha

Since 1.0
