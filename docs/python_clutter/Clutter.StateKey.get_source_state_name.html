get_source_state_name

@returns(utf8)
def get_source_state_name()

Retrieves the name of the source state of the @state_key

if this is the generic state key for the given property when transitioning to the target state. The returned string is owned by the Clutter.StateKey and should never be modified or freed

Returns :

the name of the source state for this key, or None

Since 1.4
