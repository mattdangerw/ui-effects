queue_redraw_with_clip

@accepts(cairo.RectangleInt)
@returns(none)
def queue_redraw_with_clip(clip)

Queues a redraw on @self limited to a specific, actor-relative rectangular area.

If @clip is None this function is equivalent to clutter_actor_queue_redraw().

clip :

a rectangular clip region, or None

Since 1.10
