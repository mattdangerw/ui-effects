get_states

@returns(<list>)
def get_states()

Gets a list of all the state names managed by this Clutter.State.

GList of state names. The contents of the returned GList are owned by the Clutter.State and should not be modified or freed. Use g_list_free() to free the resources allocated by the returned list when done using it

Returns :

a newly allocated
