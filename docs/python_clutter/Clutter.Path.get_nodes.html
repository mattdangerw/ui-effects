get_nodes

@returns(<list>)
def get_nodes()

Returns a GSList of Clutter.PathNode<!-- -->s. The list should be freed with g_slist_free(). The nodes are owned by the path and should not be freed. Altering the path may cause the nodes in the list to become invalid so you should copy them if you want to keep the list.

list of nodes in the path.

Returns :

a
