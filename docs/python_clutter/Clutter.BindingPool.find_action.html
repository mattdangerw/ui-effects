find_action

@accepts(guint, Clutter.ModifierType)
@returns(utf8)
def find_action(key_val, modifiers)

Retrieves the name of the action matching the given key symbol and modifiers bitmask.

returned string is owned by the binding pool and should never be modified or freed

key_val :

a key symbol

modifiers :

a bitmask for the modifiers

Returns :

the name of the action, if found, or None. The
