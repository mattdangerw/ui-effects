get_pixel_format

@returns(Cogl.PixelFormat)
def get_pixel_format()

Retrieves the pixel format used by @texture. This is equivalent to:

|[ handle = clutter_texture_get_pixel_format (texture);

if (handle != COGL_INVALID_HANDLE) format = cogl_texture_get_format (handle); ]|

Returns :

a CoglPixelFormat value

Since 1.0
