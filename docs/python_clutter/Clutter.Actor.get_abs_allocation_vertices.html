get_abs_allocation_vertices

@accepts([Clutter.Vertex])
@returns(none)
def get_abs_allocation_vertices(verts)

Calculates the transformed screen coordinates of the four corners of the actor; the returned vertices relate to the Clutter.ActorBox coordinates as follows: <itemizedlist> <listitem><para>v[0] contains (x1, y1)</para></listitem> <listitem><para>v[1] contains (x2, y1)</para></listitem> <listitem><para>v[2] contains (x1, y2)</para></listitem> <listitem><para>v[3] contains (x2, y2)</para></listitem> </itemizedlist>

verts :

Pointer to a location of an array of 4 Clutter.Vertex where to store the result.

Since 0.4
