Clutter.Script:translation-domain

"translation-domain"             utf8                : Read / Write

The translation domain, used to localize strings marked as translatable inside a UI definition.

If Clutter.Script:translation-domain is set to None, Clutter.Script will use gettext(), otherwise g_dgettext() will be used.
