get_preferred_width

@accepts(Clutter.Container, gfloat, gfloat, gfloat)
@returns(none)
def get_preferred_width(container, for_height, min_width_p, nat_width_p)

Computes the minimum and natural widths of the @container according to @manager.

See also clutter_actor_get_preferred_width()

container :

the Clutter.Container using @manager

for_height :

the height for which the width should be computed, or -1

min_width_p :

return location for the minimum width of the layout, or None

nat_width_p :

return location for the natural width of the layout, or None

Since 1.2
