set_editable

@accepts(gboolean)
@returns(none)
def set_editable(editable)

Sets whether the Clutter.Text actor should be editable.

An editable Clutter.Text with key focus set using clutter_actor_grab_key_focus() or clutter_stage_set_key_focus() will receive key events and will update its contents accordingly.

editable :

whether the Clutter.Text should be editable

Since 1.0
