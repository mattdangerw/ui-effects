get_bounds

@accepts(guint8, guint8)
@returns(none)
def get_bounds(opacity_start, opacity_end)

Gets the initial and final levels of the opacity applied by @behaviour on each actor it controls.

opacity_start :

return location for the minimum level of opacity, or None

opacity_end :

return location for the maximum level of opacity, or None

Since 0.6
